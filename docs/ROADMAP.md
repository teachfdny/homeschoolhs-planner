# HomeschoolHS Planner - Development Roadmap

## Current Version: v1.0

### v1.0 Features (Complete)
- ✅ 4-year course planner
- ✅ Course management (add/delete)
- ✅ GPA calculation
- ✅ Local data storage
- ✅ Basic transcript generation
- ✅ Export/import plans
- ✅ Responsive design
- ✅ Multi-student support (via export/import)

## v2.0 (Q3 2026) - Enhancements

### New Features
- 🎯 **Custom Grading Scales** - Support weighted GPA, 5.0 scales (for Honors/AP)
- 🎯 **Weighted GPA** - AP/IB courses add 0.5 to GPA points
- 🎯 **Requirements Checklist** - Track graduation requirements
- 🎯 **Course Recommendations** - Suggest courses based on student path
- 🎯 **Better Multi-Student** - Dropdown to switch between students
- 🎯 **PDF Export** - Download transcript as professional PDF

### Improvements
- 🔧 Better UI/UX
- 🔧 Dark mode option
- 🔧 Mobile app version
- 🔧 Keyboard shortcuts

## v3.0 (Q4 2026) - Multi-School

### New Features
- 👥 **User Accounts** - Create account, login (optional)
- 👥 **Multiple Schools** - Manage several schools in one account
- 👥 **Student Profiles** - Add photo, contact info
- 👥 **Admin Dashboard** - View all students, generate reports
- ☁️ **Cloud Backup** - Auto-save to cloud (optional)
- 📊 **Analytics Dashboard** - Track trends, GPA over time

## v4.0 (2027) - Advanced

### New Features
- 🎓 **College Prep Tools**
  - Recommended course paths
  - College requirements mapping
  - SAT/ACT tracking
  - College applications tracker

- 📚 **Advanced Tracking**
  - Dual enrollment courses
  - Test scores (SAT, ACT, AP, IB)
  - Extracurriculars
  - Leadership roles

- 📋 **Reporting**
  - Detailed grade reports
  - Parent/student portal
  - Progress reports
  - Comparison reports

## Technical Roadmap

### v2.0 Tech Stack
- Current: HTML/CSS/JS + localStorage
- Future: Add backend API (optional)

### v3.0 Tech Stack
- Frontend: React (optional, for better state management)
- Backend: Node.js + Express
- Database: PostgreSQL or Firebase
- Auth: Firebase Auth or Auth0

### v4.0 Tech Stack
- Microservices architecture
- Mobile app (React Native)
- Advanced analytics
